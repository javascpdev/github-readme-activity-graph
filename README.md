# Github Readme Activity Graph

A dynamically generated activity graph to show your GitHub activities.

| :----------------: | :-------------: | :---------------------------------------: |
| Graph Construction | **CHARTISH.JS** | <https://github.com/gionkunz/chartist-js> |
